### NODEJS-IPFS SERVER

### Environment requirement

> NodeJS: `v18.17.1` or LTS latest 
>
> Npm: `v9.8.1`

### Install Dependencies
```bash
cp .env.example .env
```
>
> Install project dependencies:
```bash
npm i
```
or
```bash
yarn
```

### Development (For Developer)

> Run:
```bash
npm start
```
> Open browser at: 
```bash
http://localhost:3000
```
